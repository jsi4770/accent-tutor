# 화면 명세 (Screen Flow Spec)

전체 흐름은 워크플로우 다이어그램을, 홈/스피킹/성적표 3개 화면의 세부 UI는 PRD 워크시트의 Q5 묘사를 기반으로 정리했다. [PRD.md](./PRD.md)와 함께 본다.

## 전체 네비게이션 플로우

```
시작 화면 (서비스 소개)
 ├─ 회원가입 (구글/네이버/카카오 소셜 로그인 연동)
 └─ 로그인 (소셜 로그인 + 자체 아이디 로그인 + ID/PW 찾기)
        ↓
 악센트 선택 (en-UK / en-AU / en-US, 추후 변경 가능)
        ↓
 레벨 테스트 (진행률 표시, 듣기+선택형, 말하기 녹음, 스킵 가능)
        ↓
 학습 목표 설정 (유학/취업/여행/일상회화 카드 선택 + 일일 목표 시간)
        ↓
 홈 화면 ──────────────┬──────────────→ 프로필 (닉네임/레벨/악센트뱃지/목표/누적학습/뱃지/발음추이)
   │                   └──────────────→ 내 정보 (계정/비번/악센트/목표/알림/로그아웃/약관 등)
   ├─→ 말하기 학습 화면 → 발음(말하기) 피드백 ─┐
   └─→ 악센트별 표현 학습 → 표현 학습 완료 요약 ─┤
                                              ↓
                                    학습/진도 기록 (연속 학습일, 탭 분류, 상세 이동)
                                              ↓
                                          복습 (발음/표현 복습 탭 분리 또는 통합 리스트)
```

하단 탭바(핵심 4탭): **홈 · 학습 이력 · 성적표 · 마이페이지**

---

## 1. 시작 화면
- 서비스 소개
- CTA: 회원가입 / 로그인

## 2. 회원가입
- 구글/네이버/카카오 소셜 로그인 연동 API

## 3. 로그인
- 소셜 로그인 연동
- 내부 자체 아이디 로그인
- ID/PW 찾기(재발급)

## 4. 악센트 선택
- 악센트별 카드(썸네일): en-UK / en-AU / en-US
- "추후 변경 가능" 안내 문구

## 5. 레벨 테스트
- 상단 진행률 표시
- 듣기+선택형 문제 (듣기 재생 버튼)
- 말하기 녹음 문제 (녹음 버튼)
- 테스트 스킵 버튼

## 6. 학습 목표 설정
- 목표 카드 선택: 유학 / 취업 / 여행 / 일상회화
- 하루 목표 학습 시간(분) 설정

---

## 7. 홈 화면 ★핵심 화면 1

**상단 영역 — 프로필 & 국가 선택 토글**
- 좌측: 사용자 이름("김민수 님") + 소형 프로필 이미지
- 우측: **목적지 국가 선택 토글스위치** [UK] [AU] [US] — 활성 탭은 딥블루(Deep Blue) 브랜드 컬러
- 사용자 이름/레벨/악센트 뱃지 표시

**중앙 영역 — 상황별 학습 카드 (세로 스크롤)**
- 유저의 학습 목적(유학/워홀/비즈니스 등)에 맞춘 카드형 콘텐츠 세션
- 예: "동네 펍(Local Pub) - 영국인 동기들과 첫 소셜라이징" 카드
  - 상황 설명 텍스트 (예: "펍에서 친구들과 모였습니다. 맥주를 주문(My round)하고 대화 리듬에 자연스럽게 녹아들어 보세요.")
  - 예상 싱크로율 점수(별 아이콘)
- "오늘의 학습" 리스트: 각 항목에 제목 · 예상 소요시간 · 완료 여부 체크

**하단 영역 — CTA & 내비게이션**
- [실전 스피킹 시작하기] 버튼 (둥근 사각형, 넓게 배치)
- 하단 탭바: 홈 / 학습 이력 / 성적표 / 마이페이지

## 8. 인터랙티브 실시간 스피킹 및 AI 피드백 화면 ★핵심 화면 2

**상단 영역 — 진행 상황 & 오디오 상태**
- 대화 진행도 프로그레스 바
- 소형 마이크 아이콘 + "오디오 스트리밍 안심 모드" 문구 (소음 감지 시 깜빡이며 백엔드 필터링 중임을 시각적으로 안내)

**중앙 영역 — 인터랙티브 대화창 & 추천 팝업**
- 메신저 대화방 UI: AI 원어민 튜터의 TTS 질문 말풍선 → 유저 발화 STT 텍스트
- **현지 표현 동적 추천 팝업**: 유저 말풍선 위에 소프트 옐로우 팝업이 동적으로 등장
  - 예: "💡 문법은 완벽해요! 하지만 영국 현지에서는 drugstore 대신 chemist's라고 표현하면 런던 현지인 느낌을 200% 살릴 수 있어요. 다시 한번 영국식 리듬으로 말해볼까요?"

**하단 영역 — 음성 녹음 UI**
- 큼직한 원형 [마이크 버튼]
- 버튼을 누르면 주파수 파형 그래프 대신, 부드러운 네이비 컬러 웨이브 애니메이션으로 발음·운율 데이터 버퍼링

→ 완료 시 **발음(말하기) 피드백** 화면으로 이동
- 전체 점수(발음/억양/유창성 등)
- 문장별 파형 비교 (원어민 vs 사용자 발화)
- 교정 포인트 요약

## 9. 악센트별 표현 학습
- 동일 의미 표현을 악센트별로 나란히 비교
- 각 표현별 발음 재생 버튼, 예문/사용 맥락 설명
- 선택한 악센트 표현 저장/북마크 기능
- 다음 표현 이동 버튼

→ 완료 시 **표현 학습 완료 요약** 화면으로 이동
- 이번 세션 학습 표현 개수, 리스트
- 맞춘 개수/헷갈렸던 표현 표시
- 북마크한 표현 확인

## 10. 스토리텔링형 지역 싱크로율 성적표 ★핵심 화면 3

**상단 영역 — 스토리텔링 대시보드**
- 도착 도시 랜드마크 일러스트 + 헤드라인 (예: "[사용자명] 님은 지금 '런던 출근 3일 차', 영국인 싱크로율 58%")
- 누적 성취도 원형 게이지 바

**중앙 영역 — 기본 발음 vs 악센트 지표 (분리 표시)**
- **지표 1: 탄탄한 기본 발음 (Standard Accuracy)** — 억양과 무관하게 딕션 자체의 명료도, 막대그래프
- **지표 2: 영국식 악센트 싱크로율 (en-UK Prosody)** — 고저차·리듬감(운율) 정밀 추출 점수, 지표 1 바로 아래 배치해 두 지표를 한눈에 비교

**하단 영역 — Grow & 다음 챌린지 제안**
- [기본기 다지며 다시 도전하기 (런던 튜브 타기 상황극)] 배너 버튼

---

## 11. 학습/진도 기록
- 연속 학습일, 누적 학습 시간
- 발음/표현 기록 탭 분류
- 리스트 아이템 클릭 → 해당 세션의 발음 피드백 상세로 이동

## 12. 복습
- 발음 복습 / 표현 복습 탭 분리 또는 "오늘의 복습 리스트"로 통합 노출 (설계 결정 필요 — 몰입 vs 유형별 학습)

## 13. 프로필
- 프로필/닉네임, 현재 레벨, 선택한 악센트 뱃지, 학습 목표 표시
- 누적 학습 시간/일, 획득 뱃지/업적, 발음 점수 추이 요약

## 14. 내 정보
- 계정 정보, 비밀번호 변경, 악센트 변경, 학습 목표 재설정
- 알림 설정, 로그아웃/회원탈퇴, 이용약관/개인정보관리방침/고객센터 문의

---

## 결정 필요 사항 (Open Questions)
- 복습 화면: 발음/표현 복습을 탭으로 분리할지, 하나의 리스트로 통합할지.
- 레벨 테스트 스킵 시 기본 난이도/악센트 추천 로직.
- 소음 필터링 실패 시(필터링 후에도 인식 불가) 재시도 UX 세부 동작.
